📁 FICHIERS DISPONIBLES
================================================================================

1. tess3.ps1
   └─ Le script PowerShell principal (ameliore)

2. Lancer_EchoThyr.bat  ⭐ RECOMMANDE POUR DEBUT
   └─ Double-cliquez pour lancer le script avec fenetre visible
   └─ Vous verrez tous les messages et notifications
   └─ Arret : Ctrl+C ou fermez la fenetre

3. Lancer_EchoThyr_Silencieux.vbs
   └─ Lance le script en arriere-plan (sans fenetre)
   └─ Utile pour demarrage automatique au demarrage Windows
   └─ Les notifications sonores/visuelles fonctionnent toujours

4. Arreter_EchoThyr.bat
   └─ Arrete tous les processus EchoThyr en arriere-plan
   └─ Utilisez si vous avez lance la version silencieuse

5. Modele_Echo.docx
   └─ Template Word pour les comptes rendus


📋 DEMARRAGE RAPIDE
================================================================================

PREMIERE UTILISATION :
1. Double-cliquez sur "Lancer_EchoThyr.bat"
2. Verifiez que tous les chemins sont valides (affichage vert)
3. Laissez la fenetre ouverte
4. Deposez un dossier patient dans C:\EchoThyr\export
5. Attendez le double BEEP et la banniere verte !

🆘 PROBLEMES COURANTS
================================================================================

"Execution Policy" erreur :
→ Ouvrez PowerShell en admin
→ Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

Tesseract non trouve :
→ Verifiez l'installation dans C:\Program Files\Tesseract-OCR\
→ Reinstallez Tesseract si necessaire

Template non trouve :
→ Assurez-vous que Modele_Echo.docx existe
→ Chemin exact : C:\EchoThyr\Modele_Echo.docx

Aucun CR genere :
→ Verifiez les logs dans C:\EchoThyr\logs\
→ Format du dossier : "NOM Prenom" (espace entre les deux)
→ Images doivent etre en .jpg
